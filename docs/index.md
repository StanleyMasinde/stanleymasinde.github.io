# 👋 Welcome

Welcome to my collection of open source projects — ideas crafted with purpose, code written with care.

## 🛠 Featured Projects

### 🔗 [Lnky](https://lnky.netlify.app)  
[![GitHub Repo](https://img.shields.io/badge/GitHub-lnky-181717?logo=github)](https://github.com/stanleymasinde/lnky)

> **Lnky** strips trackers from URLs before you share them — because your privacy shouldn't come with fine print.

- ✅ Cleans invasive query params  
- 🌐 Supports platforms like YouTube, Facebook, Amazon  
- ⚙️ Built for performance and extensibility

### 📝 [Notes](https://notes-n.netlify.app)  
[![GitHub Repo](https://img.shields.io/badge/GitHub-notes-181717?logo=github)](https://github.com/StanleyMasinde/notes)

> **Notes** is an offline-first note-taking app — open your browser and start writing.

- ✍️ Instant note-taking, no setup required  
- 💾 Auto-saves locally with IndexedDB  
- 🧩 Inspired by the sleek UX of Samsung Notes  
- 🚫 No backend, no accounts, no tracking

### 🛢️ [DB Mirror](https://github.com/StanleyWorks/db_mirror)  
[![GitHub Repo](https://img.shields.io/badge/GitHub-db__mirror-181717?logo=github)](https://github.com/StanleyWorks/db_mirror)

> **DB Mirror** lets you clone your production database to staging or local environments effortlessly.

- ⚙️ Built in Rust for speed and safety  
- 🔁 Syncs production to staging/local with minimal hassle  
- 💡 Ideal for faster dev/testing without polluting prod  
- 🔐 No cloud dependencies, runs locally

### 🌐 [URL Expander](https://github.com/StanleyMasinde/url_expander)  
[![GitHub Repo](https://img.shields.io/badge/GitHub-url__expander-181717?logo=github)](https://github.com/StanleyMasinde/url_expander)

> **URL Expander** is the Rust-powered backend for Lnky — it takes short URLs and reveals their full destination.

- 🦀 Written in Rust for speed and reliability  
- 🔍 Expands links from bit.ly, t.co, youtu.be, and others  
- 🔒 Can be integrated into privacy-focused workflows  
- ⚙️ Headless API component for internal use in Lnky
